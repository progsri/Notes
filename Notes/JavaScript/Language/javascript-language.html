Object.assign

Object.assign is used to create a new object. Lets create a new object todo. 

                    todo = {
                        name : "test",
                        place : "place",
                        type : "type"
                    }
The below code returns a new object. It takes the first argument {} as the base object, then appends the second argument to it and then replace third argument in the newly created object. 

                    // Returns a New Object
                    todo_new = Object.assign({}, todo, {type:"NONE"});
                    console.log(todo_new)
                

                    // Returns a New Object
                    todo_new_append = Object.assign({j:"tt"}, todo, {type:"NONE"});
                    console.log(todo_new_append)
