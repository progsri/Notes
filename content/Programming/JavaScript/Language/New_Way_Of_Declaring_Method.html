New Way of Declaring a Method

The below code is the old fashioned way of declaring a method. 


               function todosOld(state = [] , action){
                    console.log("old way of declaration");
                }

                todosOld();

Output 


                 old way of declaration

The below code is the new way of declaring a method. 


               const todosNew = ( state = [] , action) => {
                    console.log("new way of declaration");
                };

                todosNew();

Output 


                 new way of declaration
