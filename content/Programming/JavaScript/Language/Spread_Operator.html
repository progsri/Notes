Spread Operator 


                    testarray = [1,2];
                    newarray = testarray.concat(3);  // returns a new array
                    console.log(newarray);

Output 


                   [ 1, 2, 3 ]

The above code can be written as follows. 


                    testarray = [1,2];
                    newarray2 = [...testarray, 3];  // returns a new array
                    console.log(newarray2);

Output 


                   [ 1, 2, 3 ]
