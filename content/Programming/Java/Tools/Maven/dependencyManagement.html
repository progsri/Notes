dependencyManagement

dependencyManagement is used when more than one module is using the same dependency. It is always part of the parent. We mark all the common dependenices under dependencyManagement tag and then in the module ( child ) we will have to explicity mention the dependency we need under the dependencies .

It helps to keep all the dependencies and the version same in all the modules. Without this each module might use a different version of the dependency as they can mention the version explicity.

dependencyManagement does not download the dependencies, the modules should explicity mention the dependency they need under dependencies
