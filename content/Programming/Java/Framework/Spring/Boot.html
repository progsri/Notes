Boot

It is a lite weigth service.

Can be deployed as a jar or war.

Use Spring starter to create a Spring boot. 


 @SpringBootApplication
    public class Application {

      public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
      }
    }

@SpringBootApplication is composed of :

  • @EnableAutoConfiguration
  • @ComponentScan
  • @Configuration

Package scanning or Component scanning

Properties

By default Spring looks into /main/resources/application.properties , to read or inject just use @Value(${keyFromPropertiesFile})

Static files

These are served from /main/resources/static .

Dependenices

Use Spring starter to create a Spring boot or find dependencies, these are spring boot starter which help in finding compatible one. We will always need spring-boot-starter-parent and then comes spring-boot-starter-web, spring-boot-starter-seurity, spring-boot-starter-jpa.

Maven Plugin

Use spring-boot-maven-plugin. This gives mvn spring-boot:run

Tools

In development, having spring-boot-devtools as maven dependency will auto reload when there is a code change.
