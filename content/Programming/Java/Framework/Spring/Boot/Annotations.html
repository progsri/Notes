Annotations

@ConditionalOnBean by type

Will create the bean annotated with @ConditionalOnBean only if Spring container finds the list of bean(s) of the type mentioned in the argument

CondClassPresent bean will only be created if the beans of type ClassI and ClassH are created 


@Component
public class ClassI {

    public ClassI(){
        System.out.println("Class I : Constructor");
    }
}

@Component
public class ClassH {

    public ClassI(){
        System.out.println("Class H : Constructor");
    }
}

@Configuration
@ConditionalOnBean({ClassI.class, ClassH.class})
public class CondClassPresent {

    CondClassPresent(){
        System.out.println("CondClassPresent :: constructor");
    }
}

@ConditionalOnBean by name of the bean

Will create the bean annotated with @ConditionalOnBean only if Spring container finds the list of bean(s) names mentioned in the argument

CondClassPresent bean will only be created if the beans of name bean1 and bean2 created 


@Component("bean1")
public class ClassI {

    public ClassI(){
        System.out.println("Class I : Constructor");
    }
}

@Component("bean2")
public class ClassH {

    public ClassI(){
        System.out.println("Class H : Constructor");
    }
}

@Configuration
@ConditionalOnBean(name = {"bean1","bean2"})
public class CondClassPresent {

    CondClassPresent(){
        System.out.println("CondClassPresent :: constructor");
    }
}

@ConditionalOnProperty based on property existence

CondPropertyClass bean would be created only if there a property prefix-part.name-part= 


@Configuration
@ConditionalOnProperty(prefix = "prefix-part", name = "name-part")
public class CondPropertyClass {

    CondPropertyClass(){
        System.out.println("CondPropertyClass :: constructor");
    }
}

@ConditionalOnProperty based on a specific value for property

CondPropertyClass bean would be created only if there a property prefix-part.name-part with a value specific-value 


@Configuration
@ConditionalOnProperty(prefix = "prefix-part", name = "name-part", havingValue = "specific-value")
public class CondPropertyClass {

    CondPropertyClass(){
        System.out.println("CondPropertyClass :: constructor");
    }
}
