Actuator

To get actuator we need dependency spring-boot-started-actuator , just by adding this we need all the endpoints automatically, these endpoints start with /actuator

Monitor the application

Health check

/actuator/health returns a json {"status":"UP" } which can be modified to our application. Let say we can make a call to DB and if we get the response then we can send a diffrent json back.

Metrics of the application

/actuator/info reads the application.properties and anything that starts with info such info.app.name or info.app.version would be picked up

Other

By default only /health and /info are exposed, to expose all of them add this management.endpoints.web.exposur.einclude=* to activate all of them.

list of all endpoints

Security for endpoints

By default all end points are secured by spring security, We can apply spring security knowledge to use single sigon or Oauth for security
