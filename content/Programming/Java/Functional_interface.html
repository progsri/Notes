java.util.function.Supplier using lambda expression

This is a one of the functional interface, Supplier takes no arguments and always returns a value. To get the return value use get(), this would return value of type T.

The lambda expression such as () -> customMethod() does not take any argument and return a value of type T. So it always return a Supplier.

 }
