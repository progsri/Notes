Time Series

Timestamp

One of the data structure for representing single value time-series data 

import pandas as pd
a = pd.to_datetime('2019-01-15 3:45pm')
print(a)

b = pd.to_datetime('2019/01/12')
print(b)

c = pd.to_datetime('01/12/2019', dayfirst=True)
print(c)

print(type(c))  # Type of c


2019-01-15 15:45:00
2019-01-12 00:00:00
2019-12-01 00:00:00

class 'pandas._libs.tslibs.timestamps.Timestamp'

DatetimeIndex

One of the data structure for representing lists time-series data 

import pandas as pd

a = pd.to_datetime(['2018-01-05', '7/8/1952', 'Oct 10, 1995'])
print(a)
print(type(a))


DatetimeIndex(['2018-01-05', '1952-07-08', '1995-10-10'], dtype='datetime64[ns]', freq=None)
class 'pandas.core.indexes.datetimes.DatetimeIndex'
