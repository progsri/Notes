Attribute binding

v-bind

Binds a attribute in this case src of img to the data key baseImage in vue, so if the value of the baseImage changes vue automatically updates the image as well.

shorthand for v-bind is :

index.html 
    

        <div id="app">
        
    <img v-bind:src="baseImage" width="10%"></img>
    <h3>{{product}}</h3>
    
    </div>
main.js 

        var app = new Vue({
            el: '#app',
            data: {
                product: 'socks',
                baseImage: './grey-sock.jpeg'
            }
        }
        )
