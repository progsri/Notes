Two way binding

v-model

v-model binds the data object property to the user input of the input element and also binds the user input to the vue's data object property.

index.html 
    

    <div id="app">
    <input v-model="userInput" type="text">
    <br>
    {{userInput}}
    </div>
main.js 

    var app = new Vue({
        el: '#app',
        data: {
            userInput: "initial value"
        }
    }  
    )
