Conditional rendering

v-show

v-show simply shows if element if the expresion evalutes to true. This does not add or remove the element from the dom, it simply uses the element show / hide properties to control the display. Better than v-if

index.html 
    
            <div id="app">

            <img v-bind:src="baseImage" width="10%"></img>
    
            <h3>{{product}}</h3>
    
            <div v-show="stockQuantity > 10 ">In stock</div>
            <div v-show="stockQuantity > 1 && stockQuantity < 10 ">Out of stock soon</div>
            <div v-show="stockQuantity == 0 ">Out of stock</div>
        
        </div>
main.js 

        var app = new Vue({
            el: '#app',
            data: {
                product: 'socks',
                baseImage: './grey-sock.jpeg',
                stockQuantity: 9
            }
        }
        )
