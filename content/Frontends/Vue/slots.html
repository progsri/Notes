Slots

Slots can used to pass specific html elements from the parent.

Style can be changed/ added from the child component. ( Might be changing )

If slot is not passed, we can have the default content to display in the child template. 


import Vue from 'vue'
import App from './App.vue'

new Vue({
  el: '#app',
  render: h => h(App)
})
