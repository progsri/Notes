Router

This does internal routing instead of going to the server for each request by default, however this behaviour can be changed to go to the server for every request by mode:'history' , otherwise the default would be mode:'hash'.

Router makes our application the SINGLE PAGE APPLICATION

Installation

Router does not come by default, it has to be installed using npm install vue-router. 

import Vue from "vue";
import VueRouter from "vue-router";
import App from "./App.vue";
import Home from "./views/Home.vue";
import Login from "./views/Login.vue";

Vue.use(VueRouter); //  add vue router module to vue framework

const router = new VueRouter({
  routes: [{ path: "/", component: Home }, { path: "/login", component: Login }]
});

new Vue({
  router,
  render: h => h(App)
}).$mount("#app");
