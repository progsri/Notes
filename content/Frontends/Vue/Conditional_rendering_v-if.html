Conditional rendering

v-if

v-if manipulates the dom so that based on the expression outcome it can add or remove the entire html element, It is not a good option in terms of performance to add or element many times, in such cases prefer v-show

"" can be used to evaluate an expression as seen below

index.html 
    
            <div id="app">

            <img v-bind:src="baseImage" width="10%"></img>
    
            <h3>{{product}}</h3>
    
            <div v-if="stockQuantity > 10 ">In stock</div>
            <div v-else-if="stockQuantity > 0 && stockQuantity < 10 ">out of stock soon</div>
            <div v-else>Out of stock</div>
    
        </div>
main.js 

        var app = new Vue({
            el: '#app',
            data: {
                product: 'socks',
                baseImage: './grey-sock.jpeg',
                stockQuantity: 9
            }
        }
        )
