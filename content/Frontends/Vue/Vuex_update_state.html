Vuex update state

Synchronously via mutations

To update synchronously use mutations via this.$store.commit("MUTATION_NAME","some dynamic value")

The mutation method in the vuex store always gets the state as the first argument.

Asynchronously vis actions

To update asynchronously use actions via this.$store.dispatch("ACTION_NAME")

This could be used to call API in background and update state when we get the response, use computed in components to reflect the update state.
