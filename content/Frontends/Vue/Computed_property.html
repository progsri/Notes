Computed property

Adding a property to computed object of vue, it gives the same effect as adding to data except that it would be cached. so if the properties on which computed property does not change then vue pulls it from cache.

index.html 
    

    <div id="app">
    <div > {{cachedProperty}} </div >
    </div>
main.js 

        var app = new Vue({
            el: '#app',
            data: {
                product: "socks",
                company: "Universal"
            },
            computed: {
                cachedProperty() {
                    return this.company + " " + this.product
                }
            }
        }
        )
