Component

Vue.component is used to make html components reusable, these components have their own html, javascript and css.They also have their own Vue objects data, methods, computed.

Component name is used as html element to embed in the parent html

Parent component talks to the child component using props

Why in child component the data object should be a function.

index.html 
    

    <div id="app">
    <div>{{fromParent}}</div>
    <product :tochild="fromParentToChild"></product>
    </div>
main.js 

    Vue.component('product', {
        props: ['tochild'],
        template: `
        <div>
        <button @click="changeMe">{{fromComponent}}</button>
        <br>
            {{tochild}}
            </div>
        `,
        data() {
            return {
                fromComponent: "child component data"
            }
        },
        methods: {
            changeMe() {
                this.fromComponent = "changed"
            }
        },
        computed: {
    
        }
    })
    
    var app = new Vue({
        el: '#app',
        data: {
            fromParent: "parent component data",
            fromParentToChild: "from parent to Child "
        }
    }
    )
