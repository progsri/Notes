Filters

Lets you apply a function to the value.

Filters can be chained

Local Filter

This is only visible to that component the filter is defined at. 



import Vue from 'vue'
import App from './App.vue'

new Vue({
  el: '#app',
  render: h => h(App)
})
App.vue 

<template>
  <div>
    Value
    <input v-model="value" type="text" />
    <p>After filter :: {{value |  upper}}  </p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value: ""
    };
  },
  filters:{
    upper: function(entry) {
      return entry.toUpperCase();
    }
  }
};
</script>

Global Filter

These filters are visible to all the components


import Vue from "vue";
import App from "./App.vue";

Vue.filter("lower", function(entry) {
  return entry.toLowerCase();
});

new Vue({
  el: "#app",
  render: h => h(App)
});
App.vue 

<template>
  <div>
    Value
    <input v-model="value" type="text" />
    <p>After filter :: {{value | lower}}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value: ""
    };
  }
};
</script>

Using computed property instead of filter

computed property can also be used instead of filters.

In the below only if this.value changes then the computed property will be recalculated

App.vue 

<template>
  <div>
    Value
    <input v-model="value" type="text" />
    <p>After filter :: {{upper}}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value: ""
    };
  },
  computed: {
    upper: function() {
      return this.value.toUpperCase();
    }
  }
};
</script>
