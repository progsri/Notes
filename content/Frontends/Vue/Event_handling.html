Event Handling

v-on

Binds to a event handler method. When a event (click / hover / keypress etc ) is trigger on the dom, we attach a method to process that event, in vue that method is under the methods.

In order to access data from the event method, we need to use this as keys under data are registered as instance variables.

shorthand for v-on: is @

index.html 
    

        <div id="app">
        
    <img v-bind:src="baseImage" width="10%"></img>
    <h3>{{product}}</h3>
    
    </div>
main.js 

        var app = new Vue({
            el: '#app',
            data: {
                product: 'socks',
                baseImage: './grey-sock.jpeg',
                cart: 0
            },
            methods: {
                add: function () {
                    this.cart = this.cart + 1  //this refers to the instance value of the app object which has data and that 
                                                //has cart
                }
            }
        }
        )
