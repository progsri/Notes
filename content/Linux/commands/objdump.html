objdump

Example

Here we are using objdump to disassemble the ls command's binary code. It shows the binary code in hexa decimal on left side and their assembly code on the right side. Both the side are same just one is in assembly which is human readable and the other is the actual code that the machine understands which is nothing but the assembly code. 

 
objdump -d /bin/ls

 
/bin/ls:     file format elf64-x86-64
