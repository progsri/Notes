Layer 2

also called as Data link layer

Layer 2 receives the data from layer 1 ( ethernet cables, wifi radio signals, fiber optics ) and reassembles them based on the layer 1 it is interacting with. For example if a network card is used with a ethernet cable then the ethernet cable uses IEEE 802.3 protocol, based on this the Layer 2 device in this case network card knows what is the start and end of the frame. Frame is the unit used at the Layer 2 level which has all the information it received for one packet.

Layer 2 deals with MAC address . MAC address are created by the manufacture, it stands for Media Access control. MAC is considered as a physical address as it does not change compared to a IP address that can change for a machine.

Examples of L2 devices are switch, Network card, Router.

The main goal of this layer is to deliver messages from one machine to another machine. These machines communicate using the layer 1 suchas a cable. So the intelligence lies in the layer 2 device with layer 1 just used as medium to transfer the data.
