object pod

pod is the smallest unit of kubernetes, It can have one ore more containers. Generally related containers are grouped together as pod.

configuration

Kind indicates the type of Object, it can be a service, pod , replicaController etc. name indicates the name of the pod lables are key value pair that are used by other objects such as service to relate their config to the pod. containers as the pod is made of container(s), has the name of the container, image and the port the container needs.

Config file client-pod.yaml to create a pod. 

apiVersion: v1
kind: Pod
metadata:
  name: client-pod
  labels:
    component: web
spec:
  containers:
    - name: client
      image: stephengrider/multi-client
      ports:
        - containerPort: 3000

Applying the configuration kubectl apply -f client-pod.yaml 


pod/client-pod created

Get pods kubectl get pods 


NAME         READY   STATUS    RESTARTS   AGE
client-pod   1/1     Running   0          2m52s

RESTARTS 0 indicate the pod was never restarted
