object service

this object is used for networking.There are different type of services:

  • Cluster Ip
  • NodePort : used only in local and never in production
  • Load balancer
  • Ingress

Configuration 


apiVersion: v1
kind: Service
metadata:
  name: client-node-port
spec:
  type: NodePort
  ports:
    - port: 3050
      targetPort: 3000
      nodePort: 31515
  selector:
    component: web
