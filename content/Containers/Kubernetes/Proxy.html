Proxy

kubectl proxy gives connection to your kubernetes cluster,it creates tunnel to the configured cluster and opens the access over the 8081 port thru which we can call the Kubernetes Rest API . Some of the endpoints are. This connection is only between the system on which the kubectl proxy is running and the cluster.

  • http://127.0.0.1:8001/api
  • http://127.0.0.1:8001/api/v1
  • http://127.0.0.1:8001/api/v1/pods
  • http://127.0.0.1:8001/api/v1/namespaces

The ApI provides the information which the command line binary kubectl provides or the web interface provides thru UI.
